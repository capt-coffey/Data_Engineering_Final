# Census API Setup

## Overview
 This data ingestion uses the **U.S. Census Bureau ACS 5-Year Estimates API** to retrieve socio-economic data for the St. Louis region.

## API Key Registration
1. Request a free API key from: [https://api.census.gov/data/key_signup.html](https://api.census.gov/data/key_signup.html)
2. Email verification.
3. Key activation.

## Environment Configuration
Store the key in local development (.env) file to keep it private and safe.
